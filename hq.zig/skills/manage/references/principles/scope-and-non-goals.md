# Scope And Non Goals

Status: partial

## Meaning

Every active lane should have one primary scope.
Anything outside that scope should be named explicitly as a non-goal.

Tool repair and target work should be separated whenever possible.

## Rules

- declare the only active scope before substantive work begins
- declare explicit non-goals
- keep tooling repair separate from target-task execution
- if scope changes, say so and restate the new boundary
