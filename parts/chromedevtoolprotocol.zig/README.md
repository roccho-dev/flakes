# chromedevtoolprotocol.zig

Curated Zig module for Chrome DevTools Protocol (CDP) operations.

This repo is a self-managed fork/slice derived from prior `zchrome` work, packaged
as a reusable Zig module and tested via contract-focused RED/GREEN cases.

Within this workspace it lives under `parts/chromedevtoolprotocol.zig` so the
repo's Zig components and dependencies share the same top-level layout.
